About The Site:

This site features a final project for ITIS 6177 - System Integration that implements Microsoft Azures's "Computer Vision" API which is an AI service that analyzes content in images and video.


Implemented Features:

  • Text Extraction
    • From Microsoft Azure's Page On The Computer Vision API: "Extract printed and handwritten text from images and documents with mixed languages and writing styles." On this site, this feature allows you to submit different image links with readable text and the system extracts that text for you.

  • Image Description
    • From Microsoft Azure's Page On The Computer Vision API About The Image Understanding Feature: "Pull from a rich ontology of more than 10,000 concepts and objects to generate value from your visual assets." I specifically implemented this feature to describe images based off of links that are submited. On this site, this feature allows you to submit different image liks and the system will describe those images for you. The system will also provide some tags that factored into its decision for the images description.
